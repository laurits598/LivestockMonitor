# Geo-Fencing application

This is a simple geo-fencing application built with **FastAPI**

## Project structure

farmer/
- Geo.py 
- index.html 
- README.md 

---
### Software required 

- Python 3.12.2
- fastapi
- uvicorn
- shapely
- pydantic
- aiofiles
--- 

#### Install all dependencies

```bash
pip install fastapi uvicorn shapely pydantic aiofiles
```

##### How to run the code 

```bash
fastapi dev Geo.py
```

###### Open your browser at

- http://127.0.0.1:8000/ui/
